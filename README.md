# 🚀 NX Club Web


---

## 🌟 About the Project

✨ **NX Club Web** is a modern and responsive website built for a technical club to showcase its **events, workshops, team members, and activities** in a structured and attractive way.

It is designed with a clean UI and simple navigation to help students easily explore club information and stay updated.

---

## 🎯 Features

- 🏠 Home page with club introduction  
- 📖 About section explaining vision & mission  
- 🎉 Events showcase (past & upcoming)  
- 🛠️ Workshops & learning sessions  
- 🖼️ Gallery for memories & highlights  
- 👨‍💻 Team section (Leads, Members, Interviewers)  
- 📩 Contact page for communication  

---

## 🛠️ Tech Stack

- 🧱 HTML5 – Structure  
- 🎨 CSS3 – Styling & Layout  
- 📱 Responsive Design (Flexbox / Grid)
